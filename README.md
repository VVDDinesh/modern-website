## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

## Customization

The website uses CSS variables for easy customization. Edit the `:root` variables in `styles.css` to change colors, spacing, and other design elements.

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #ec4899;
    /* ... more variables */
}
```
